Static

pong — two players, real time over the internet. Create a game and share the link (or 4-letter code); WebRTC pairs you peer-to-peer with ~frame latency — no server at all, even though this site is static. Mouse, touch, or W/S.

crush — dodge the falling blocks. A plain HTML file in this repo, served by GitHub Pages.

life — Conway's Game of Life: born-on-3, survive-on-2/3 on a toroidal grid. Click/drag to draw cells, space to pause, c clears, r randomizes. ~1 KB.

oregon trail — the 1978 Creative Computing classic (public domain). Vanilla JS, ~21ms to playable; beat a wasm-C port (31ms) for the spot — both byte-identical to the Python original over 600 scripted games. The Pyodide build it replaces took 3.1s. Type fast when it says BANG.

Live (self-hosted)

unsuckify — tell it what you suck at, get roasted into being better (Claude runs on the box).

the live page — served from my own machine by a through a Cloudflare tunnel, kept alive across reboots. It has server state: type a message there and it lands as a file on the box. (Easter egg: type crush on the page.)
